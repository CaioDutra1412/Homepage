Lista de Exercícios JavaScript


Número 1

Faça uma função para exibir uma mensagem.


Número 2

Faça uma função para exibir uma mensagem e solicitar que o usuário digite o nome.


Número 3

Faça uma função para solicitar que o usuário digite o nome, exiba este nome em uma caixa de alerta e depois no html.


Número 4

Acrescente faça uma função que exiba um campo onde o usuário deverá digitar algum valor, deixe claro o intervalo e verifique se o valor está no intervalo solicitado.


Número 5

Solicite ao usuário que digite 2 valores quaisquer, faça uma função para comparar qual é o maior valor.


Número 6

Crie um aplicativo que receba uma temperatura qualquer em Fahrenheit e apresente seu correspondente em Celsius por meio de uma Função. Para o calculo utilize a seguinte fórmula: Celsius=5.0/9.0*(F-32)


Número 7

Imprima a tabela de tabuada de um determinado número:
Deve-se exibir o número cuja tabela queremos mostrar;
O valor desse número multiplicado pelos números de 1 a 10;
Um contador iniciando com valor 1 e sendo incrementado até chegar a 10;
